Students:

Students registered can get the information regarding their attendance, view assignment topics, form teams for the assignment, get notified about the results and if selected, get the information about the project and the guide assigned to them.

Teachers:

The website provides an easy to use interface for the teachers to manage the entire process of the internship program right from bootcamp till the end of project. They can use the website to evaluate the team assignments as well as the projects.
